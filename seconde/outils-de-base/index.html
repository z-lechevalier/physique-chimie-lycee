Outils de base en physique-chimie

Conversions, formules, matériel — Seconde

Ces ressources sont conçues pour une classe inversée : la capsule vidéo se regarde avant la séance, le temps de classe étant consacré aux activités et aux exercices. Les quiz ne servent pas à évaluer mais à apprendre — ils sont faits pour être refaits.

Les quiz sont des fichiers HTML autonomes : ils s'ouvrent dans un navigateur, sans installation ni compte, et n'enregistrent aucun résultat. Les QCM de vérification de visionnage sont des fichiers XML, à importer dans Pronote ou dans Moodle.
