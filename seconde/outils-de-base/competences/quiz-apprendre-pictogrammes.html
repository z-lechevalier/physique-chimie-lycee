Reconstitue le tableau des pictogrammes de danger. Sur chaque ligne, une case t'est donnée (le pictogramme, la signification, les précautions OU les exemples) : à toi de placer les trois autres à l'aide de la banque d'étiquettes en bas de l'écran (onglets Pictogrammes / Significations / Précautions / Exemples). Clique sur une case vide : le bon onglet s'ouvre, puis clique sur l'étiquette qui convient. Pour reprendre une étiquette mal placée, clique dessus dans le tableau.
Quand tout est rempli, clique sur « Valider le tableau » : chaque ligne compte pour 1 point (ses 3 cases doivent être justes). Au 2ᵉ essai, seules les étiquettes des lignes fausses retournent dans les banques.
Tentatives Éléa illimitées — 2 essais par ligne dans le quiz. Relancer l'activité remet tout à zéro et remélange le tableau.
