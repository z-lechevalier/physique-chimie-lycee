Reconstitue le grand tableau de la verrerie : pour chaque photo (colonne de gauche), place le bon nom, le bon schéma et la bonne utilisation.
Les étiquettes sont rangées dans la banque en bas de l'écran, classée en trois onglets (Noms / Schémas / Utilisations). Clique sur une case vide du tableau : l'onglet correspondant s'ouvre. Clique ensuite sur l'étiquette qui convient pour la poser. Pour reprendre une étiquette mal placée, clique simplement dessus dans le tableau.
Quand tout est rempli, clique sur « Valider le tableau » : chaque ligne compte pour 1 point (ses 3 cases doivent être justes). Au 2ᵉ essai, seules les étiquettes des lignes fausses retournent dans les banques.
Tentatives Éléa illimitées — 2 essais par ligne dans le quiz. Relancer l'activité remet tout à zéro et remélange le tableau.
