Complète le grand tableau des préfixes. Sur chaque ligne, une case t'est donnée : à toi de placer les deux autres. Clique sur une étiquette dans une banque, puis sur la case vide où tu veux la poser (tu peux aussi cliquer d'abord sur la case). Pour reprendre une étiquette mal placée, clique simplement dessus dans le tableau. Quand le tableau est complet, clique sur « Valider le tableau » : chaque ligne compte pour 1 point (ses 2 cases doivent être justes). Au 2ᵉ essai, seules les étiquettes des lignes fausses retournent dans les banques.
Le bouton « ↺ Recommencer » remet tout à zéro et remélange le tableau.
